# Aakarsh Narang — Portfolio

Welcome to my personal portfolio. This project serves as a comprehensive showcase of my journey as a Software Engineer, highlighting my expertise in backend infrastructure, applied AI, and developer community leadership.

## 🛠 Tech Stack

The portfolio is engineered with a focus on performance, interactivity, and modern design aesthetics:

* **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6+)
* **Animation Engine:** Custom Boids Flocking Algorithm for the interactive particle swarm cursor
* **Typography:** Space Grotesk, Inter, and Space Mono (served via Google Fonts)
* **Iconography:** SimpleIcons

## ✨ Key Features

* **Terminal-Style Preloader:** A 4.5-second technical boot sequence featuring simulated system logs and a reactive progress bar.
* **Boids Swarm Cursor:** An organic, physics-based particle system that exhibits flocking behavior and naturally settles when the cursor is idle.
* **Dynamic Hover States:** Tech stack components that trigger vibrant, brand-specific radial gradients across the background upon interaction.
* **Performance-First:** Utilizes efficient `requestAnimationFrame` loops for smooth 60fps animations with minimal overhead.
* **Responsive Layout:** A fluid, fully optimized design that adapts seamlessly from high-resolution displays to mobile devices.
